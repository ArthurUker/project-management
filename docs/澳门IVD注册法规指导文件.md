## 一、更新结论摘要

## 二、本项目法规定位

### 2.1 产品属性初步判断

本项目为：

> **10项呼吸道病原体检测用微流控体外诊断系统**

可能包含以下组成：

- 体外诊断试剂 / 试剂卡 / 微流控芯片
- 配套检测仪器
- 数据采集、结果判读、报告输出软件
- 样本处理或采样相关耗材
- 校准品、质控品或内控材料

因此，本项目不应仅按一般医疗器械处理，而应按：

> **体外诊断医疗器械 / 体外诊断试剂 + 配套仪器 + 软件组件的组合型 IVD 系统**

进行澳门注册策略设计。

### 2.2 注册路径初步判断

基于现有文件，本项目大概率属于需要**注册**的体外诊断医疗器械，而不是简单备案产品。

当前应优先完成：

1. 依据第 1/ISAF/2026 号批示进行分类判定；
2. 判断是否属于第 12/ISAF/2026 号备案路径；
3. 若不属于备案路径，则按第 7/ISAF/2026 号批示准备注册资料；
4. 同时按第 2/ISAF/2026 号与第 5/ISAF/2026 号判断临床评价路径；
5. 按第 20/ISAF/2026 号和第 16/ISAF/2026 号准备生产质量体系资料。

## 四、注册申报总体路线

### 4.1 推荐申报主线

```text
产品定义与预期用途确认
    ↓
IVD 属性确认与分类判定（第1号）
    ↓
注册/备案路径判断（第1号、第12号）
    ↓
注册资料目录建立（第7号）
    ↓
技术文件准备
    ├─ 产品技术要求
    ├─ 设计开发资料
    ├─ 风险管理资料
    ├─ 分析性能验证资料
    ├─ 稳定性资料
    ├─ 软件资料
    ├─ 标签说明书
    └─ QMS/生产质量资料
    ↓
临床评价路径判断（第2号、第5号）
    ├─ 豁免临床评价
    ├─ 同品种比对 + 文献/临床数据
    └─ 临床试验（第3号、第4号）
    ↓
注册卷宗终审与提交（第7号）
    ↓
技术审评、补充资料、批准
    ↓
上市后变更、续期、QMS维护（第10号、第11号、第16号、第20号）
```

### 4.2 不建议的申报假设

在目前资料下，不建议直接假设：

- 产品可直接备案；
- 产品可豁免临床评价；
- 仅凭内地/NMPA资料即可完全满足澳门要求；
- 软件资料只需简单说明；
- QMS 只提交 ISO 13485 证书即可；
- 标签说明书可沿用内地版本不做澳门本地化审核。

## 六、针对第20号批示的 IVD 专项要求

第 20/ISAF/2026 号批示是本次更新中对项目影响最大的新增文件之一。其核心意义是：

> IVD 试剂不能只满足通用医疗器械 QMS，还必须满足体外诊断试剂生产质量管理的专项要求。

### 6.1 人员要求

项目需确认并留档：

- 生产、技术、质量人员具备医学、检验学、生物学、免疫学、药学或相关专业知识；
- 洁净区人员接受卫生、微生物学基础知识、洁净作业培训；
- 生产、检验、物料、清洁、维修、废弃物处理人员接受与产品特性匹配的培训；
- 直接接触原材料、中间产品、成品人员至少每年体检一次；
- 患有传染性及感染性疾病人员不得直接接触相关物料或产品；
- 建立洁净工作服/无菌工作服管理制度。

### 6.2 厂房与洁净区要求

对于本项目，应根据生产工艺判断是否涉及以下操作：

- 配液
- 包被
- 分装
- 点膜
- 干燥
- 切割
- 贴膜
- 内包装
- 阳性/阴性血清处理
- 质粒、血液制品处理
- 无菌原材料或中间产品分装
- PCR/核酸扩增试剂生产或检验

如涉及免疫荧光、PCR、金标、干化学、校准品与质控品、酶类、抗原、抗体及其他活性组分，其相关生产工序通常需在**不低于100,000级洁净区**内进行。

如涉及阴性/阳性血清、质粒或血液制品处理，相关区域通常不得低于**10,000级洁净度**，且须与相邻区域保持相对负压。

如涉及无菌原材料及中间产品分装，局部操作区域须符合**100级洁净度**。

### 6.3 洁净度级别

| 洁净度级别 | ≥0.5μm 尘粒最大允许数/立方米 | ≥5μm 尘粒最大允许数/立方米 | 浮游菌/立方米 | 沉降菌/皿 |
|---|---:|---:|---:|---:|
| 100级 | 3,500 | 0 | 5 | 1 |
| 10,000级 | 350,000 | 2,000 | 100 | 3 |
| 100,000级 | 3,500,000 | 20,000 | 500 | 10 |

### 6.4 温湿度要求

如无特殊生产工艺要求，洁净室/区一般应控制：

| 参数 | 推荐控制范围 |
|---|---|
| 温度 | 18-28℃ |
| 相对湿度 | 45%-65% |

### 6.5 PCR/核酸扩增污染控制

若产品涉及 PCR 或其他核酸扩增相关试剂，应重点关注：

- 生产及检验应在独立建筑物或独立空间内进行；
- 空气不得直接联通；
- 生产和质检器具不得混用；
- 使用后须严格清洗和消毒；
- 防止扩增气溶胶造成交叉污染；
- 如涉及危险度二级及以上病原体操作，应配备生物安全柜；
- 操作区空气应过滤处理后排出；
- 过滤器性能需定期检查。

```sql
CREATE TABLE regulatory_documents (
  id UUID PRIMARY KEY DEFAULT gen_random_uuid(),

    applicability IN ('core', 'conditional', 'post_market', 'low_relevance', 'not_applicable')
```

```sql
CREATE TABLE task_regulatory_documents (
  task_id UUID NOT NULL REFERENCES tasks(id) ON DELETE CASCADE,

  PRIMARY KEY (task_id, regulatory_document_id)
);
```

```sql
ALTER TABLE tasks
```

## 十二、结论

本次 26项 ISAF/2026 批示更新后，澳门 IVD 注册申报工作应重点围绕以下六条主线推进：

1. **分类与注册路径主线**：第1号、第12号；
2. **注册卷宗主线**：第7号；
3. **临床评价主线**：第2号、第5号、第3号、第4号；
4. **标签说明书主线**：第6号；
5. **IVD生产质量体系主线**：第16号、第20号、第22号；
6. **软件与网络安全主线**：第21号。

对本项目而言，最关键的新增合规要求是：

> **第20/ISAF/2026号《体外诊断试剂生产质量管理规范》与第21/ISAF/2026号《医疗器械独立软件生产质量管理规范》应纳入主任务体系。**

因此，原 47 项任务建议升级为 **66 项任务**，并在系统中增加法规依据、适用性判断、资料包类型和法规知识库模块。
